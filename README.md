# # ✊ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript.

![Game Preview](https://via.placeholder.com/800x400?text=Rock+Paper+Scissors+Game)

## 🎮 Features

- Beautiful modern UI with glassmorphism effect
- Live score tracking (You vs Computer)
- Smooth hover animations
- Fully responsive (works perfectly on mobile & desktop)
- Reset score button
- Clean and beginner-friendly code

## 🚀 How to Play

1. Click on **Rock**, **Paper**, or **Scissors**
2. Computer will instantly pick its move
3. Winner is decided by classic rules:
   - ✊ Rock beats ✌️ Scissors
   - ✌️ Scissors beats ✋ Paper
   - ✋ Paper beats ✊ Rock

Same choice = Tie

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Modern Glassmorphism)
- Vanilla JavaScript (No frameworks)
